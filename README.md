# Cycle Bloom – Menstrual Tracker (Mobile-first)

Mobile-style UI with three tabs (Home, Calendar, Logs).  
Soft pastel look matching the original mockup.

## Run
```bash
npm install
npm start
```
## Build
```bash
npm run build
```
## Deploy (Netlify)
- Connect repo → build command `npm run build` → publish `build/`.
